## Module 1 Summary

In Module 1, I learned the basic foundations of cloud computing and AWS.

The main idea of cloud computing is the on-demand delivery of IT resources over the internet with pay-as-you-go pricing. Instead of owning and maintaining all infrastructure on-premises, businesses can use AWS resources as needed and pay only for what they consume.

I also learned about the client-server model, where a client sends a request and a server processes that request and returns a response. Cloud computing fits into this model by providing scalable server resources over the internet.

Another major topic in this module was the AWS Global Infrastructure. AWS uses Regions and Availability Zones to help businesses improve high availability and fault tolerance. A Region is a geographic area, and each Region contains multiple Availability Zones, which are distinct locations with one or more data centers.

I also learned about the AWS Shared Responsibility Model. AWS is responsible for security of the cloud, while the customer is responsible for security in the cloud. This means AWS protects the underlying infrastructure, but customers still need to secure their operating systems, applications, and data.

Finally, I learned about the main benefits of the AWS Cloud, including pay-as-you-go pricing, economies of scale, not having to guess capacity, increased speed and agility, reduced data center maintenance, and the ability to go global quickly.

## Key Points to Remember

- Cloud computing is the on-demand delivery of IT resources over the internet with pay-as-you-go pricing.
- In the client-server model, the client makes requests and the server responds.
- AWS Regions are geographic areas, and Availability Zones are distinct locations within Regions.
- AWS Global Infrastructure supports high availability and fault tolerance.
- AWS is responsible for security of the cloud.
- Customers are responsible for security in the cloud.
- Hybrid deployment combines on-premises infrastructure with cloud resources.
- One major cloud benefit is avoiding the cost and complexity of running physical data centers.
