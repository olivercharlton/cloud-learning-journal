## About

I'm using this repository to track what I'm learning as I build my foundation in cloud engineering. It includes notes, labs, weekly updates, and useful resources as I work through core topics step by step.

## Current focus

Right now I'm focused on:

- AWS Cloud Practitioner
- Linux fundamentals
- Networking basics
- Git and GitHub
- Cloud concepts and core services

## Repository structure

- `notes/` — topic notes and explanations
- `labs/` — small hands-on exercises and practice
- `weekly-updates/` — summaries of what I studied and learned each week
- `resources/` — useful links, references, and learning materials

## Goals

- Build a strong foundation in cloud engineering
- Develop consistent hands-on practice
- Document my progress publicly
- Grow from beginner to job-ready over time

## Why this repo exists

I'm creating this repository to stay organized, show my progress, and build the habit of learning in public while developing real technical skills.
